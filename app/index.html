Patterns

Atoms (UI Components) -> Compounds (Groups of components) -> Molecules (template blocks) -> Pages (groups of template blocks). Interfaces should be built using consistent Molecules.

Molecules

Molecules are collections of compounds. They can be used as templates, and define the high-level structure of Pages.

  • Media Component
  • 'Holler' Block
  • 'Three-up' Block
  • 'Slider' Block

Compounds

Compounds are collections of atoms. They can be dropped into Molecule layouts to form templates.

  • 'Title' Block
  • 'CTA' Block
  • Form Block
  • Input Block
  • Image Block

Atoms

Atoms are individual UI components. They can be combined together to make Compounds.

  • Calls to Action
  • Titles
  • Optional-Titles
  • Inputs
  • Images
