Patterns

Quarks (UI Components) -> Atoms (Groups of components) -> molecules (template blocks) -> Pages (groups of template blocks). Interfaces should be built using consistent molecules.

Molecules

Molecules are collections of Atoms. They can be used as templates, and define the high-level structure of Pages.

  • Media Component
  • 'Holler' Block
  • 'Three-up' Block
  • 'Slider' Block
  • 'Two-up' Block

Atoms

Atoms are collections of Quarks. They can be dropped into molecules to create Templates.

  • 'Title' Block
  • 'CTA' Block
  • Form Block
  • Input Block
  • Image Block

Quarks

Quarks are individual UI components. They can be combined together to make Atoms.

  • Calls to Action
  • Titles
  • Optional-Titles
  • Inputs
  • Images
  • Captions
